# VRM Version Viewer

VRMとUniVRMのバージョン見るやーつ

## 使い方

```
./vrm_version_viewer-x_x_x-amd64.exe avater.vrm
```


## test
```bash
docker compose run --rm builder bash

go run vrm_version_viewer.go VRM_FILE
```
